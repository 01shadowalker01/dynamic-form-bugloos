Fields

In Dynamic form builder, forms are constructed with fields.

Every field must have two properties: key and code type.

The key is used to retrieve value from the form later. You should be aware that the key should be unique for each field.

On the other hand, type should be one of the following values:

  • {{ field }}

Also, you can set some optional values like label, placeholder, hint and etc (See TemplateOptions interface for more).
